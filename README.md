### 몬스터를 피해 열쇠를 찾고 탈출하세요!

## 🎮 구현기능
* 캐릭터의 이동 및 기본 동작
  * 캐릭터, 몬스터 FSM 구현
  * W/A/S/D 이동 구현
<br><br/>
* 레벨 디자인
  * Stage 1 : Key 1개를 찾아서 탈출
  * Stage 2 : Key 3개를 찾아서 탈출
<br><br/>
* 충돌 처리 및 피해량 계산
  * RayCast로 Player 공격 구현
<br><br/>
* UI/UX 요소
  * Player Hp, Stamin 구현
  * 총알 갯수 표시
  * Loby, 일시정지 구현
<br><br/>
* 다양한 무기
  * HandGun1, M4, M82 구현 및 획득
<br><br/>
* 난이도 설정
  * Normal, HardCore 설정
  * HardCore 시, 몬스터 이동속도 증가
<br><br/>
* 미션 시스템
  * 현재 수행해야 할 미션 표시
<br><br/>
* AI 적 캐릭터
  * 일정 범위 안의 Player를 쫒아오는 몬스터 구현
<br><br/>
* 사운드 및 음악 효과
  * 배경음 구현
  * 총기, 획득, 몬스터 등 다양한 사운드 구현
<br><br/>
* 특수 효과 및 파티클 시스템
  * 총기 Effect, Particle System 구현
<br><br/>
* 총기 구현
  * 마우스 좌 클릭으로 총 발사
  * 총기 반동, 정확도 구현
